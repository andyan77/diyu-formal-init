## 0. 文档目的

本文件定义笛语“怎样说话”，用于约束：

* 选题标题；
* 视频口播；
* 图文正文；
* 商品介绍；
* 品牌观点；
* 门店内容；
* 评论回复；
* 活动表达；
* 内容改写和平台适配。

声纹不是一组固定句式，而是一套稳定判断。不同账号可以有不同语气，但不能失去共同的品牌人格。

# 一、声纹总定义

## 1.1 一句话定义

> 笛语的表达应当真实、自然、有审美判断，能够说明选择与取舍，温和但不含糊，专业但不端着。

**状态：已确认战略基线。**

## 1.2 声纹核心

笛语说话时应同时具备五个特征：

| 特征  | 具体含义                |
| --- | ------------------- |
| 真实  | 不编造商品、人物、顾客和经营事实    |
| 具体  | 尽量落到人、动作、场景、商品和使用结果 |
| 有判断 | 不只描述“好看”，还说明为什么成立   |
| 有边界 | 承认不适合谁、牺牲什么、不能解决什么  |
| 可接近 | 不用行业黑话压人，不把审美说成优越感  |

# 二、笛语说话的基本姿态

## 2.1 面对消费者

笛语不是教育消费者，而是帮助消费者判断。

推荐姿态：

* “这件适合怎样的人。”
* “它解决的主要问题是什么。”
* “你可以怎样判断自己是否需要。”
* “它有优点，也有明确取舍。”
* “不同家庭成员可以有不同答案。”

避免姿态：

* “你必须这样穿。”
* “懂审美的人都会选。”
* “不买就是错过。”
* “所有妈妈都需要。”
* “任何人穿都好看。”

## 2.2 面对儿童

笛语把儿童看作独立的人，而不是内容道具。

表达应关注：

* 身体是否自在；
* 动作是否方便；
* 穿着是否符合成长阶段；
* 孩子自身的气质和选择；
* 衣服怎样服务真实生活。

不得强调：

* 成人化吸引力；
* 取悦他人；
* 过度成熟；
* 身材评判；
* 把儿童作为父母审美成绩。

## 2.3 面对家庭关系

笛语不把家庭理解为完全相同，而理解为彼此有关、各自成立。

推荐表达：

* 不必穿成一样，也能看出是一家人；
* 联系可以来自色彩、轮廓、材质或气质；
* 家庭成员之间可以有呼应，也应保留差异；
* 家庭审美不是一个人对所有人的控制。

## 2.4 面对商品

笛语不把商品写成“没有缺点的答案”。

每次商品表达优先回答：

1. 这件商品解决什么问题；
2. 适合怎样的身体、场景和穿衣目标；
3. 它最重要的优势是什么；
4. 为了这个优势，它牺牲了什么；
5. 什么人或什么场景不建议选择；
6. 哪些结论来自真实商品资料；
7. 哪些只是搭配或表达建议。

# 三、表达强度范围

| 维度 | 笛语标准范围         | 禁止漂移         |
| -- | -------------- | ------------ |
| 情绪 | 有生活温度，但不过度煽情   | 苦情、强行感动、虚构冲突 |
| 专业 | 能解释依据，但不堆术语    | 伪专业、无证据技术结论  |
| 审美 | 有明确判断，但不傲慢     | 审美优越、贬低普通消费者 |
| 销售 | 可以明确推荐，但必须有条件  | 逼单、恐吓、虚假稀缺   |
| 幽默 | 生活化、轻微自嘲       | 嘲讽顾客、羞辱身材    |
| 锋利 | 可以反对行业套路       | 攻击具体品牌、群体或个人 |
| 感性 | 可以写家庭、成长、失败和欢喜 | 编造人生事实、虚假鸡汤  |
| 潮流 | 可以使用当下语言       | 生硬追梗、平台黑话堆积  |

# 四、句子与语言习惯

## 4.1 推荐句子结构

笛语优先使用以下结构：

### 结构一：场景—问题—判断

> 早晚温差大时，孩子真正需要的不是更厚，而是方便穿脱、活动时不会束缚的叠穿结构。

### 结构二：优点—代价—适用对象

> 这类宽松轮廓能让动作更自在，但不会强调利落腰线，更适合重视松弛感的人。

### 结构三：观察—解释—建议

> 很多家庭搭配显得用力，不是颜色太多，而是每个人都在抢同一个视觉重点。保留一个主角，其他人做轻呼应，会自然很多。

### 结构四：承认差异

> 有人需要衣服替自己表达，有人只希望穿得舒服得体，这两种需求都成立。

### 结构五：具体反选

> 这件更适合干爽天气和短时间户外活动；长时间高温运动，不建议因为造型好看而勉强选择。

## 4.2 句式偏好

建议：

* 短句与中等长度句交替；
* 一个段落只讲一个主要判断；
* 先说人和问题，再说商品；
* 多使用“因为、所以、但、如果、尤其”建立逻辑；
* 少使用连续感叹号；
* 不用大量抽象形容词叠加。

## 4.3 常用表达方向

以下是表达方向，不是固定口号：

* 各自有型；
* 自然地走在一起；
* 不必完全一样；
* 真实生活；
* 穿得像自己；
* 给动作留空间；
* 选择比推荐更重要；
* 适合谁，也要说清不适合谁；
* 有设计，但不牺牲生活；
* 让家庭有联系，也保留每个人；
* 商品有优点，也有取舍；
* 好看需要回到真实场景。

# 五、禁用或限制表达

## 5.1 原则性禁用词

下列表达原则上禁用：

* 闭眼入；
* 谁穿谁好看；
* 所有人都适合；
* 不买后悔；
* 全网都在抢；
* 妈妈必买；
* 绝绝子；
* 高级到骨子里；
* 贵妇感；
* 秒杀大牌；
* 穿上立刻瘦十斤；
* 黄黑皮救星；
* 小个子天菜；
* 微胖福音；
* 显白神器；
* 永不过时；
* 零束缚；
* 绝对不起球；
* 绝对不变形；
* 百分之百安全；
* 设计师倾注灵魂；
* 每一个细节都经过无数次打磨。

并非因为这些词永远不能出现，而是它们通常：

* 缺少事实支持；
* 过度承诺；
* 制造焦虑；
* 用套话替代专业判断；
* 容易让笛语失去真实感。

## 5.2 限制使用的词

以下词可使用，但必须提供具体解释：

| 词语 | 使用条件              |
| -- | ----------------- |
| 高级 | 说明来自面料、比例、色彩还是细节  |
| 显瘦 | 说明视觉机制和适用身体，不能普遍化 |
| 舒适 | 有面料、版型、动作或试穿依据    |
| 百搭 | 说明可搭配的具体品类和场景     |
| 经典 | 说明结构为何具有稳定性       |
| 潮流 | 说明趋势来源及如何被笛语转译    |
| 质感 | 指明触感、光泽、垂坠或结构     |
| 功能 | 必须有商品资料或测试依据      |
| 环保 | 必须有材料、认证或供应链依据    |
| 原创 | 必须有权属或研发依据        |

# 六、五类内容产品的语言差异

## P1：真实穿衣问题与购买决策

语言特征：

* 先讲问题；
* 提供判断条件；
* 允许多个答案；
* 明确何时不选；
* 少用品牌自夸。

示例：

> 孩子活动量大时，外套首先要看肩背和袖窿，不是只看尺码数字。静态合身，不代表跑跳时自在。

## P2：商品真相与专业取舍

语言特征：

* 信息密度较高；
* 区分硬事实和专业解释；
* 强调优势与代价；
* 不包装不存在的设计原意。

示例：

> 这件的重点不是“更显瘦”，而是用相对清晰的肩线稳定上半身比例。它会比完全落肩的款式更利落，但松弛感也会少一些。

## P3：品牌或账号观点与生活叙事

语言特征：

* 有人物立场；
* 可以感性；
* 必须有真实生活种子；
* 不为卖货强行转折。

示例：

> 做品牌以后，我越来越能接受一件事：一家人不必在每件事上取得一致。衣服也是。真正舒服的关系，不是看起来一样，而是允许彼此不同。

## P4：一线问题回应与本地生活

语言特征：

* 更直接；
* 更口语化；
* 来自真实门店和当地场景；
* 可以带时间、天气和商圈信息；
* 不夸大普遍性。

示例：

> 今天店里被问得最多的是早晚怎么给孩子加衣。现在这个温度，厚外套还早，能随时脱下来的薄层更实用。

## P5：造型灵感与视觉叙事

语言特征：

* 文字更少；
* 留白更多；
* 强调视觉关系；
* 不用空洞诗意掩盖造型逻辑。

示例：

> 同一种蓝，不同的明度。
> 各自穿自己的，站在一起自然有了联系。

# 七、不同账号的声纹适配

| 账号角色     | 主要语言特征          | 不得越界          |
| -------- | --------------- | ------------- |
| 品牌官方号    | 稳定、清晰、代表品牌立场    | 不编个人经历        |
| 创始人账号    | 有选择、有生活经验、有反思   | 不虚构人生事实       |
| 商品研发负责人  | 专业、具体、解释结构和取舍   | 不讲未经确认的经营结论   |
| 零售与陈列负责人 | 场景化、可执行、关注空间和搭配 | 不冒充研发确认商品硬事实  |
| 内容负责人    | 观察内容、品牌表达和创作方法  | 不替品牌最高决策人修改战略 |
| 总部直营门店   | 服务感、现场感、真实问题    | 不冒充全国普遍情况     |
| 区域官方号    | 区域理解、市场观察、品牌翻译  | 不改变总部品牌原则     |
| 区域主理人    | 经营经验、本地判断、真实关系  | 不虚构总部决策       |
| 门店服务账号   | 具体、亲近、及时、问题导向   | 不发布未经授权的品牌结论  |

# 八、评论与私域回复规则

## 8.1 推荐回复方式

面对商品问题：

> 这件目前能确认的是版型和尺码信息。具体到你家孩子，建议再看身高、肩宽和活动习惯，不能只按年龄选。

面对质疑：

> 你的担心有道理。这个结论如果没有实际资料支持，我们不会把它写成商品事实。

面对不适合：

> 这款确实不是所有人都会喜欢。需要明显腰线或轻薄体感的人，可以优先看其他选择。

面对负面反馈：

> 谢谢你把具体问题说出来。我们先核对商品批次和使用情况，再给你明确答复，不先用通用话术解释。

## 8.2 禁止回复方式

* “亲亲放心购买。”
* “我们家从来没有这种问题。”
* “别人都没有反馈。”
* “可能是你使用不当。”
* “喜欢就冲。”
* “不接受任何反驳。”

# 九、声纹合格检查

每条内容发布前至少通过以下检查：

| 检查项  | 合格标准               |
| ---- | ------------------ |
| 品牌归属 | 能看出为什么属于笛语         |
| 服务对象 | 明确在帮助谁             |
| 具体程度 | 至少有一个真实场景、动作、商品或问题 |
| 判断依据 | 不是只有形容词            |
| 事实边界 | 没有新增未经确认的商品或人物事实   |
| 适用边界 | 必要时说明不适合谁          |
| 语言姿态 | 不说教、不羞辱、不制造焦虑      |
| 销售强度 | 推荐有条件，不使用强迫式表达     |
| 儿童边界 | 不成人化、不工具化          |
| 账号权限 | 发声内容符合账号角色         |

出现以下任一情况，应退回修改：

* 连续使用三个以上空泛赞美词；
* 把搭配建议写成商品事实；
* 把个别顾客反馈写成普遍结论；
* 为了转化制造身材、年龄或育儿焦虑；
* 无依据使用绝对化承诺；
* 内容更换品牌名称后依然成立，缺少品牌辨识度。

## 本文件结论

笛语声纹最重要的辨识度不是某几个词，而是稳定做到：

> 不只告诉消费者什么好看，而是帮助人理解为什么成立、适合谁、需要怎样取舍。
