## 0. 文档性质

本文件定义笛语内容主要服务的非个人化受众模型，回答：

* 内容主要写给谁；
* 他们处于什么生活阶段；
* 面临哪些真实问题；
* 在什么场景下需要笛语；
* 喜欢怎样的表达；
* 反感怎样的表达；
* P1—P5分别应为他们提供什么价值。

AudienceProfile不是顾客名单，不是CRM画像，也不对应某一个真实个人。

本文件不得保存：

* 姓名；
* 手机号；
* 家庭住址；
* 订单明细；
* 个人消费记录；
* 儿童身份信息；
* 可识别的个人行为轨迹。

# 一、核心受众定义

## 1.1 核心受众

> 笛语的核心受众，是处于稳定家庭生活阶段、重视个人审美和家庭关系，希望每位家庭成员既保有自己、又能自然协调的家庭审美组织者。

第一沟通对象通常是女性，但笛语最终服务的不是单一女性消费者，而是家庭中的：

* 女性成员；
* 男性成员；
* 儿童；
* 家庭共同生活关系。

## 1.2 核心年龄

| 人群        |    年龄基线 | 状态       |
| --------- | ------: | -------- |
| 家庭审美组织者   | 约28—40岁 | 已确认基线    |
| 伴侣或成年家庭成员 | 约28—45岁 | 合理补全·待确认 |
| 儿童        |  约3—12岁 | 已确认基线    |

年龄是内容规划参考，不作为排斥其他消费者的硬边界。

---

# 二、核心受众的生活阶段

核心受众通常处于以下阶段：

* 已建立稳定家庭关系；
* 育有学龄前或小学阶段儿童；
* 家庭消费逐渐由“买得到”转向“怎样选择更合适”；
* 工作、育儿、家庭和个人生活相互交叠；
* 既承担家庭消费决策，也希望保有个人身份；
* 开始重视长期使用、搭配效率和生活质感；
* 不再完全追逐流行，但也不愿放弃当代感；
* 对过度营销和空洞品牌语言的耐受度较低。

# 三、城市与市场环境

## 3.1 核心市场

笛语优先服务：

* 新一线城市；
* 二线城市；
* 强三线城市；
* 优质县域中心；
* 具有成熟商业供给和家庭消费能力的区域市场。

## 3.2 城市生活特征

这些市场中的核心受众通常具有以下特点：

* 既接受主流潮流信息，也重视本地现实生活；
* 购买行为同时受线上内容和线下门店影响；
* 对价格敏感，但并非只选择最低价；
* 重视实体试穿、搭配建议和售后服务；
* 需要商品适应通勤、接送、周末和家庭活动等多重场景；
* 对品牌是否真实、门店是否可靠、表达是否专业较为敏感。

# 四、家庭结构与关系特征

## 4.1 典型家庭结构

笛语不限定唯一家庭结构，但核心服务对象通常包括：

* 夫妻与一个孩子；
* 夫妻与两个孩子；
* 单亲家庭；
* 多代共同生活家庭；
* 由主要照护者承担审美和购买决策的家庭。

内容不得默认：

* 所有家庭都由母亲独立承担育儿；
* 所有家庭成员审美一致；
* 所有孩子性格相同；
* 所有男性都不关心穿衣；
* 所有家庭都追求整齐统一。

## 4.2 家庭关系观

笛语认为：

* 家庭不是要求每个人变得相同；
* 家庭成员可以拥有不同的审美和选择；
* 儿童不是成人审美的附属物；
* 家庭协调可以来自局部呼应，而不是完全复制；
* 穿衣可以成为理解彼此、协商差异的方式；
* 好的家庭形象，应建立在每个人都自在的前提下。

# 五、核心受众需要解决的问题

## 5.1 穿衣选择问题

核心受众经常需要判断：

* 这件衣服适不适合自己或孩子；
* 尺码应该怎样选；
* 版型是否适应真实动作；
* 面料和工艺是否值得；
* 一件商品能否进入现有衣橱；
* 在什么天气和场景下使用；
* 怎样搭配才不显得刻意；
* 家庭成员怎样协调但不穿成同款；
* 商品价格是否对应真实价值；
* 哪些商品看起来好看，但并不适合自己。

## 5.2 家庭审美问题

核心受众希望解决：

* 家庭共同出现时怎样形成协调感；
* 父母和孩子怎样保持不同年龄的表达；
* 男童、女童是否可以摆脱固定风格标签；
* 家庭搭配怎样避免过度用力；
* 怎样让孩子有选择空间；
* 怎样降低全家穿搭的决策成本；
* 怎样通过少量稳定单品形成长期关系。

## 5.3 商品理解问题

核心受众不仅需要知道“好不好”，还需要知道：

* 好在哪里；
* 为什么这样设计；
* 解决什么问题；
* 牺牲了什么；
* 适合谁；
* 不适合谁；
* 是否存在更合适的替代选择；
* 商品信息是否真实可信。

## 5.4 品牌信任问题

核心受众会持续判断：

* 品牌是否说真话；
* 门店是否真的理解商品；
* 账号是否只是卖货；
* 内容中的人物和故事是否可信；
* 不同账号是否各有真实来源；
* 品牌是否尊重儿童和普通家庭；
* 品牌是否愿意承认商品边界；
* 出现问题时是否能给出具体回应。

# 七、受众核心需要

## 7.1 功能性需要

* 商品信息准确；
* 尺码和版型容易判断；
* 适应真实动作；
* 方便搭配；
* 能够重复使用；
* 降低购买错误率；
* 降低家庭穿衣决策成本。

## 7.2 审美需要

* 有设计感但不过度；
* 有当代感但不盲目追潮流；
* 每个人可以保持独立风格；
* 家庭共同出现时自然协调；
* 儿童有儿童自身的气质；
* 不落入固定粉色、蓝色或成人缩小版套路。

## 7.3 情绪需要

* 被尊重；
* 不被说教；
* 不被制造焦虑；
* 不因身材、年龄或育儿方式受到评判；
* 能从品牌内容中获得理解；
* 能看到真实家庭生活中的不完美；
* 能保有个人身份，而不只成为某种家庭角色。

## 7.4 关系需要

* 与孩子形成尊重和协商；
* 与伴侣保留差异；
* 家庭成员之间自然联系；
* 不通过强制统一制造家庭感；
* 在共同生活中仍有个体空间。

# 八、消费判断特征

核心受众并非单纯价格导向，也不是单纯品牌导向。

其判断通常包括：

1. 商品是否适合真实生活；
2. 价格是否对应面料、工艺、设计和使用频率；
3. 是否容易搭配；
4. 是否适合现有家庭成员；
5. 是否需要额外购买很多配套商品；
6. 是否经得起多次穿着；
7. 品牌表达是否可信；
8. 门店服务是否专业；
9. 商品是否存在明显使用边界；
10. 是否因为内容焦虑而冲动购买。

笛语内容的任务不是压缩消费者判断，而是帮助其完成判断。

# 九、语言偏好

## 9.1 核心受众喜欢的表达

* 具体；
* 清楚；
* 有生活经验；
* 有专业依据；
* 不绕弯子；
* 能说明优点与缺点；
* 能给出可执行建议；
* 能承认不同人有不同答案；
* 不假装完美；
* 不把普通生活包装成虚假精致。

## 9.2 推荐语言特征

* 先讲真实问题；
* 再讲判断方法；
* 最后给出选择建议；
* 使用普通消费者能理解的语言；
* 少用抽象审美概念；
* 少用行业术语；
* 允许明确说“不建议”；
* 允许说“资料不足，暂时不能确认”。

## 9.3 不喜欢的表达

* 高高在上的审美教育；
* 连续堆砌形容词；
* 空洞品牌故事；
* 强行鸡汤；
* “所有人都适合”；
* “妈妈必须怎样”；
* 用育儿焦虑促进成交；
* 用身材羞辱制造需求；
* 虚假顾客反馈；
* 无依据的功能承诺；
* 只强调“高级、显瘦、百搭”；
* 把促销写成生活价值。

# 十、核心反感点

| 反感点    | 具体表现              |
| ------ | ----------------- |
| 强行同款   | 把完全相同当作家庭感的唯一方式   |
| 儿童成人化  | 成人姿态、妆容、情绪和性感表达   |
| 育儿说教   | 用单一标准评价父母和孩子      |
| 身材焦虑   | 遮肉、显瘦、年龄羞辱等无条件营销  |
| 虚假高级   | 用豪宅、酒店、黑金视觉掩盖商品普通 |
| 空洞专业   | 没有商品事实却堆砌面料和工艺术语  |
| 绝对承诺   | 永不起球、谁穿都好看、绝对舒适   |
| 过度促销   | 每条内容都围绕低价、稀缺和抢购   |
| 品牌自恋   | 只讲品牌多优秀，不解决消费者问题  |
| 人设造假   | 虚构创始人、员工或顾客经历     |
| 门店冒充总部 | 将局部情况表达为品牌整体结论    |
| 隐私冒犯   | 未经同意使用顾客、儿童和员工素材  |

# 十一、受众分层

## 11.1 核心层：家庭审美组织者

主要特征：

* 约28—40岁；
* 承担较多家庭穿衣和购买决策；
* 重视品质、搭配和真实使用；
* 希望家庭协调但保留差异；
* 是P1、P2、P3的主要受众。

## 11.2 关系层：成年家庭成员

主要特征：

* 可能不是主要购买者；
* 关注舒适、得体、效率和个人选择；
* 不希望被统一家庭造型支配；
* 是P1、P3、P5的重要受众。

## 11.3 成长层：儿童

儿童不是直接营销数据对象，而是商品和内容必须尊重的真实使用者。

内容应关注：

* 动作；
* 舒适；
* 成长；
* 自主选择；
* 年龄适配；
* 身体尊重。

## 11.4 本地服务层：门店周边家庭

主要特征：

* 对门店位置、服务、天气和商圈信息敏感；
* 更重视能否试穿、是否现货、服务是否真实；
* 需要具体、及时、可信的内容；
* 是P4的主要受众。

## 11.5 审美灵感层

主要特征：

* 可能暂时没有明确购买需求；
* 关注造型、色彩、关系和视觉表达；
* 通过长期内容建立品牌认知；
* 是P5的主要受众。

# 十三、不同渠道的受众关注差异

## 13.1 抖音

主要关注：

* 快速理解；
* 视觉吸引；
* 真实问题；
* 动作展示；
* 明确结论；
* 本地到店价值。

内容要求：

* 开头快速进入问题；
* 画面先建立人物和商品；
* 不使用纯口号式内容；
* 结论必须有条件。

## 13.2 小红书

主要关注：

* 搜索价值；
* 选择经验；
* 搭配方法；
* 商品细节；
* 可保存的信息。

内容要求：

* 标题明确问题；
* 正文有判断结构；
* 图片能够支撑结论；
* 允许反选和替代建议。

## 13.3 视频号

主要关注：

* 品牌可信度；
* 家庭关系；
* 人物表达；
* 真实经营；
* 可转发的价值观点。

内容要求：

* 可以稍完整地讲述；
* 适合品牌、创始人和门店角色；
* 不宜过度平台化追梗。

## 13.4 朋友圈与私域

主要关注：

* 是否与自己有关；
* 是否真实；
* 门店是否可靠；
* 是否值得咨询或到店；
* 是否能解决近期问题。

内容要求：

* 更具体；
* 更及时；
* 更像真实服务；
* 不连续刷屏；
* 不使用群发式叫卖口吻。

# 十四、受众数据边界

## 14.1 可以保存的受众知识

系统可以保存：

* 非个人化生活阶段；
* 典型场景；
* 常见问题；
* 语言偏好；
* 反感点；
* 区域共性；
* 内容反馈趋势；
* 匿名化评论主题；
* 高频咨询问题。

## 14.2 不应进入内容知识库的数据

禁止保存或调用：

* 顾客姓名和手机号；
* 具体家庭住址；
* 可识别的儿童身份；
* 订单和付款明细；
* 私人聊天原文；
* 未经授权的顾客照片；
* 精确个人消费能力；
* 私人健康信息；
* 家庭关系隐私；
* 对具体个人的敏感推断。

## 14.3 评论和问题的使用方式

顾客问题可以被整理为：

> 某类消费者在某类场景下提出的常见问题。

不得整理为：

> 某位具体顾客的完整身份、家庭背景和消费记录。

---

# 十五、内容生成最低受众要求

每条内容至少需要明确以下四项中的三项：

1. 写给哪类人；
2. 发生在什么场景；
3. 对方正在解决什么问题；
4. 看完后可以形成什么判断。

不合格内容包括：

* 只展示商品，没有受众问题；
* 只讲品牌观点，不知道与谁有关；
* 只写审美形容词，没有使用场景；
* 只追热点，不知道服务什么人；
* 更换品牌后对任何受众都同样成立。

| 评审项  | 问题          |

| 价值明确 | 内容提供了什么判断   |
| 语言适配 | 是否符合该人群理解习惯 |
| 情绪边界 | 是否制造焦虑或羞辱   |
| 家庭边界 | 是否尊重成员差异    |
| 儿童边界 | 是否尊重儿童主体性   |
| 数据边界 | 是否暴露个人信息    |
| 品牌关系 | 是否体现笛语的人群理解 |

## 本批结论

笛语的核心受众不是简单的“宝妈”或“童装消费者”，而是：

> 希望家庭中的每个人都能穿得像自己，同时又愿意为共同生活建立自然联系的人。

笛语内容不应通过定义消费者来限制消费者，而应通过理解其生活矛盾，帮助其完成更真实、更理性、更有审美判断的选择。
