## 0. 文档目的

本文件定义笛语内容视觉应当呈现的共同世界。

它不是完整VI手册，不决定商标尺寸、印刷色值和包装规范；它主要约束：

* 人物怎样出现；
* 商品怎样被看见；
* 儿童怎样被拍摄；
* 家庭关系怎样被表达；
* 场景、色彩、动作和镜头怎样组合；
* P5怎样制造属于笛语的造型想象。

## 当前判断基于首批26条视频样本。文件夹中包含10条MOV和16条MP4素材，可作为笛语现有内容形态与视觉特征的观察依据。

# 一、视觉基调总定义

## 1.1 一句话定义

> 笛语的视觉应让人物先成立、衣服自然进入生活，并通过色彩、轮廓、动作和气质建立人与人之间不刻意的联系。

## 1.2 核心视觉气质

笛语整体视觉应保持：

* 自然；
* 独立；
* 轻松；
* 有造型感；
* 有生活空间；
* 有当代感；
* 不幼稚；
* 不成人化；
* 不影楼化；
* 不依赖奢华场景；
* 不把人物变成静态衣架。

# 二、首批素材所呈现的视觉方向

从现有样本中，可以提炼出以下视觉方向：

| 方向      | 主要特征              | 品牌价值       |
| ------- | ----------------- | ---------- |
| 街头休闲    | 宽松轮廓、运动元素、城市或简洁背景 | 表达儿童独立和当代感 |
| 自然户外    | 草地、道路、树林、自然光      | 强化真实动作和生活感 |
| 轻学院     | 层次、衬衫、针织、秩序感      | 提供克制和清晰气质  |
| 轻时装     | 比例、廓形、细节、姿态       | 建立审美辨识度    |
| 图文拼贴    | 多画面组合、局部细节、节奏剪辑   | 适合展示造型关系   |
| 单人造型    | 强调一个人物的完整状态       | 体现个体独立     |
| 双人或多人关系 | 通过差异和呼应形成联系       | 体现家庭审美关系   |

以上方向可以并存，但每条内容需要有一个主导方向，不能把所有风格同时堆在一个作品中。

# 三、人物视觉原则

## 3.1 人物先于商品成立

画面首先应让观众感受到：

* 这是怎样的人；
* 他或她处在怎样的状态；
* 衣服怎样参与这个状态。

不应只剩下：

* 商品正面展示；
* 机械转圈；
* 统一摆拍；
* 与人物无关的空洞造型。

## 3.2 儿童视觉原则

儿童应被呈现为正在成长、活动和感受世界的人。

推荐：

* 走、跑、转身、坐下、观察、玩耍；
* 自然表情；
* 与环境发生关系；
* 保留儿童自身节奏；
* 展现衣服在动作中的状态。

禁止：

* 成人化妆容；
* 性感化姿态；
* 过度成熟的情绪表演；
* 以身材为视觉卖点；
* 模仿成人奢侈广告；
* 强迫统一表情和统一动作；
* 把儿童拍成父母炫耀品。

## 3.3 成人视觉原则

成人内容应避免两种极端：

* 只剩下功能性和日常感，没有审美；
* 只剩下高冷姿态，与真实家庭生活脱离。

推荐呈现：

* 真实但有整理的状态；
* 有生活身份；
* 有个体气质；
* 可以与儿童自然同框；
* 不刻意扮演“完美父母”。

# 四、家庭关系的视觉表达

## 4.1 关系不是复制

家庭成员不必：

* 穿完全相同的款式；
* 使用完全相同的颜色；
* 保持完全相同的造型强度；
* 做完全相同的动作。

## 4.2 建立联系的五种方式

### 方式一：色彩呼应

使用同色系、近似色、局部共同色或明度关系。

### 方式二：轮廓呼应

通过宽松、利落、圆润、修长等轮廓建立共同气质。

### 方式三：材质呼应

通过牛仔、针织、棉感、光泽或纹理建立联系。

### 方式四：细节呼应

通过帽子、鞋、包、图案、条纹或局部装饰形成关系。

### 方式五：情绪与动作呼应

不同服装也可以通过步伐、视线、互动和空间位置形成家庭感。

## 4.3 视觉关系层级

一个家庭画面中建议明确：

* 一个视觉主角；
* 一个或两个次级角色；
* 一个共同联系点；
* 每个人自己的差异点。

避免每个人都成为最强视觉重点。

# 五、商品视觉原则

## 5.1 必须看见的内容

根据内容任务，至少清楚呈现以下三类信息中的两类：

1. 整体轮廓；
2. 动作状态；
3. 面料或细节。

P2商品内容应尽量同时呈现三类。

## 5.2 商品展示顺序

推荐顺序：

1. 人物与整体状态；
2. 商品轮廓；
3. 动作中的变化；
4. 局部细节；
5. 搭配关系；
6. 适用场景。

避免一开始就长时间展示吊牌、Logo或静态细节。

## 5.3 不得通过画面制造虚假结论

画面不得暗示：

* 未经证明的防水、防污、保暖等功能；
* 不存在的高级材质；
* 实际不具备的垂坠或挺括；
* 经过过度修图的颜色和版型；
* 所有人穿着都具有同一效果。

# 六、场景基线

## 6.1 推荐场景族

| 场景族     | 适用内容            |
| ------- | --------------- |
| 城市街道    | 街头、通勤、区域生活      |
| 社区周边    | 门店、本地家庭、真实日常    |
| 公园和自然   | 儿童活动、家庭出行、轻户外   |
| 家庭空间    | 生活叙事、穿搭准备、成长记录  |
| 门店空间    | P2、P4、搭配服务、试穿问题 |
| 简洁棚拍    | 商品结构、造型实验、视觉大片  |
| 商圈和城市地标 | 区域账号、本地生活       |
| 工作和研发空间 | 真实研发、选品和内容过程    |

## 6.2 场景选择原则

场景必须满足至少一个条件：

* 能解释人物状态；
* 能解释商品用途；
* 能增强品牌气质；
* 能证明真实经营或生活情境。

仅仅“漂亮”不足以成为场景选择理由。

## 6.3 不推荐场景

* 与品牌定位无关的豪宅；
* 过度奢华酒店；
* 纯粹模仿奢侈品牌的棚景；
* 无法解释的异国街景；
* 过度梦幻儿童布景；
* 大量装饰压过人物和商品；
* 为追求流量生硬加入的热点场景。

---

# 七、色彩基线

## 7.1 当前内容色彩方向

正式标准色尚待VI资料确认，但内容视觉可以优先使用：

* 自然中性色；
* 低至中等饱和度色彩；
* 清晰但不过度刺激的局部亮色；
* 牛仔蓝、灰、米白、卡其、棕、黑白等稳定底色；
* 根据季节加入有控制的绿、红、黄、蓝。

**状态：合理补全·待品牌方确认。**

## 7.2 色彩关系原则

推荐：

* 一个主色；
* 一个辅助色；
* 一个小面积强调色；
* 家庭多人画面保持色彩联系，但不复制。

避免：

* 同时出现过多高饱和颜色；
* 每件商品都成为色彩中心；
* 通过滤镜大幅改变商品真实颜色；
* 儿童内容使用过度甜腻的固定粉蓝体系；
* 用黑金配色机械制造“高级”。

---

# 八、构图与镜头基线

## 8.1 构图原则

推荐：

* 给人物动作留空间；
* 全身、半身、局部形成信息层次；
* 人物与环境保持真实比例；
* 多人画面保留彼此独立空间；
* 允许不完全居中；
* 允许局部遮挡和自然动态。

避免：

* 所有人整齐站成一排；
* 每个镜头都正面看镜头；
* 商品永远处于画面正中央；
* 为突出腿长进行过度低角度拍摄；
* 过度修饰儿童身体比例。

## 8.2 镜头节奏

P5短视频建议：

* 15—30秒为常用范围；
* 3—8个主要镜头；
* 每个镜头承担明确任务；
* 不依赖大量快速切换制造“潮流”；
* 先建立整体气质，再展示细节；
* 保留一两个完整动作镜头。

具体时长可根据平台和内容合同调整。

## 8.3 推荐镜头类型

* 环境建立镜头；
* 人物走入画面；
* 全身轮廓；
* 转身或侧面动作；
* 手部、衣摆、面料细节；
* 多人关系镜头；
* 自然互动；
* 离开画面或动作收束。

---

# 九、光线与后期

## 9.1 光线

推荐：

* 自然光；
* 柔和方向光；
* 能保留面料细节的光线；
* 不过度消除阴影；
* 肤色和商品颜色可信。

避免：

* 过度磨皮；
* 过度曝光；
* 冷暖色严重偏移；
* 用强光制造不存在的材质光泽；
* 儿童皮肤塑料化。

## 9.2 后期原则

后期可以：

* 调整曝光；
* 校正色温；
* 统一画面节奏；
* 清理无关隐私信息；
* 进行必要构图裁切；
* 统一笛语品牌标识。

后期不得：

* 改变商品实际颜色；
* 改变版型轮廓；
* 修改人物身材；
* 增加不存在的商品细节；
* 将普通场景包装成虚假经营现场；
* 通过AI生成不存在的真人品牌事实。

---

# 十、品牌标识使用原则

所有正式内容只体现笛语品牌身份。

素材进入正式使用前，应检查：

* 品牌名称是否统一为笛语；
* 水印是否统一；
* 片头片尾是否统一；
* 字体和Logo是否符合正式规范；
* 画面中是否存在需要替换或清理的非笛语品牌文字；
* 封面、标题和字幕是否只使用笛语当前口径。

内部素材治理统一使用：

> 品牌标识状态：待统一为笛语品牌标识。

不得在用户可见或内部标签中记录其他品牌身份。

# 十一、P5视觉叙事模型

## 11.1 P5核心目标

P5不是简单展示商品，而是建立：

> 只有笛语的品牌关系观、商品组合和人物状态才能成立的视觉世界。

## 11.2 五种基础叙事模型

### 模型一：各自成立

每个人使用不同造型，画面通过一个共同元素建立联系。

适合：

* 家庭多人造型；
* 男女童组合；
* 不同年龄成员。

### 模型二：同一种情绪，不同的衣服

成员衣服不同，但共同呈现松弛、活力、安静或利落等气质。

适合：

* 品牌视觉片；
* 季节主题；
* 系列发布。

### 模型三：从一个人到一群人

先建立单个人物，再逐渐出现其他家庭成员，最后形成关系。

适合：

* 短视频；
* 品牌故事；
* 系列造型。

### 模型四：衣服跟随动作

用走、跑、转身、坐下等动作展示版型和生活适应性。

适合：

* 童装；
* 轻户外；
* P1与P5结合。

### 模型五：生活中的视觉发现

从门店、街道、家庭或旅行中的一个真实细节发展成造型画面。

适合：

* 本地生活；
* 创始人内容；
* 区域账号；
* 品牌生活叙事。

---

# 十二、P5内容最低合格标准

每条P5内容必须满足：

1. 至少有一个明确视觉主题；
2. 人物或商品至少有一个视觉主角；
3. 至少存在一种可解释的造型关系；
4. 人物动作符合真实身份；
5. 儿童不成人化；
6. 商品颜色和轮廓可信；
7. 品牌标识统一为笛语；
8. 画面不依赖无关奢华场景；
9. 不只是“漂亮”，还应体现品牌关系观；
10. 更换为其他品牌后不能完全无损成立。

---

# 十三、视觉禁区

以下情况应直接退回：

* 全家复制同一套服装；
* 儿童摆出成人性感姿态；
* 过度磨皮、瘦身或拉长身体；
* 商品颜色严重失真；
* 画面暗示未经证明的功能；
* 使用与笛语无关的品牌标识；
* 奢华场景压过人物和商品；
* 照搬其他品牌经典视觉；
* 仅靠滤镜和音乐制造气氛；
* 画面没有任何可解释的造型关系；
* 将儿童当作成人造型缩小版；
* 为追求冲击力破坏真实穿着状态。

# 十四、视觉评审表

| 评审维度 | 问题             |
| ---- | -------------- |
| 人物成立 | 人物是否具有独立状态     |
| 商品可信 | 商品颜色、轮廓和细节是否真实 |
| 动作自然 | 动作是否符合人物和场景    |
| 关系成立 | 多人之间是否有自然联系    |
| 品牌辨识 | 能否看出属于笛语       |
| 儿童边界 | 是否尊重儿童成长和身体    |
| 场景合理 | 场景是否服务内容       |
| 视觉重点 | 是否有清晰主次        |
| 后期克制 | 是否存在过度修饰       |
| 标识统一 | 是否只体现笛语        |
| P5价值 | 是否制造了有意义的审美想象  |

## 本文件结论

笛语视觉最核心的识别不是某一种固定滤镜、色彩或拍摄模板，而是：

> 每个人先成为自己，再通过衣服、动作和生活空间，自然地与其他人发生联系。
