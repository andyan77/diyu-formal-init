## 关闭线

用户通过真实软件入口输入 P1 黄金包 1 的弱种子或等价新种子，系统不是回放固定答案，而是基于当前品牌、账号和最小激活资产形成一份完整文字成品；用户自然修改后产生可读取的新版本，并可主动保存。UI—应用 API 契约可审阅且实现一致，本地黄金回归入口可重复运行。现场凭据不可用时只把真实模型验收标为外部阻断，测试替身不得冒充真实生成通过。
