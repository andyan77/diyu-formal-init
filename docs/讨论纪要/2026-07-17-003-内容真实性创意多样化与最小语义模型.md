# 2026-07-17 内容真实性、创意多样化与最小语义模型

> **后续修订提示（2026-07-18）**：本纪要建立的“硬事实/专业判断/策略/经历/创意”分级仍有价值，但“真实经历不能补写”“生活包必须具有外部真实性记录”“品牌替换后仍成立即失败”等口径过于绝对。当前改为：用户声明的人设、经历和事件是可直接使用但未经外部核验的内容前提；系统可按 `ContentProduct` 的增益政策生成结构、条件化专业解释和受控情景演绎，同时隔离用户原始种子与系统派生，避免派生内容升级为硬事实。品牌替换测试只是组合专属性评测信号，不是单项硬阻断。详见 [2026-07-18-023](2026-07-18-023-讨论文档对齐与内容编译哲学.md)。本文件保留为 D-013—D-016 的早期形成过程，冲突处不再代表现行口径。

## 总判断

用户对“AI 内容趋同、讲课腔、缺乏真情实感和无法满足不同企业”的担忧是项目 P0 风险，而不是生成提示词层的小问题。

研究已经观察到类似风险：一项短篇故事实验发现，生成式 AI 可以提高个体作品评价，却降低作品集合的差异性。该结果不能直接等同于中国自媒体运营，但足以说明“每条看起来更顺，同时全体越来越像”是现实风险，而不是主观焦虑。

项目的内容竞争力不应定义为“模型能写多少”，而应定义为：

> 能否把可信事实、品牌独有判断、普通人的真实生活、当下媒介语境和有意识的创意选择组合成可持续、可辨认、不过度重复的内容。

任何 LLM、知识图谱或静态知识库都不能凭空生产真情实感。若没有持续进入系统的门店观察、店员声音、真实顾客问题、城市生活片段和品牌真实取舍，模型只能对互联网平均表达做更流畅的重组。

## 证据不会天然压制创意，错误的绑定方式才会

不应要求每句话都带引用。应以“内容中的陈述性质”为单位分级：

| 陈述性质 | 约束 | 创作自由 |
|---|---|---|
| 硬事实 | 必须关联权威来源、版本和有效期；参数、价格、库存、人物履历等不可改写为另一事实 | 可改变表达顺序和修辞，不能改变含义 |
| 专业判断 | 必须有适用条件、例外和责任专家/审核人 | 可以换角色和场景表达，不得说成无条件真理 |
| 企业策略 | 必须有制定权限、适用组织、版本和审批 | 可在授权范围内本地化，不得越权覆盖 |
| 真实生活/人物经历 | 必须有真实生活记录及相应隐私、肖像、声音或版权许可 | 可忠实剪裁和叙事化，不能补写不存在的经历 |
| 创意表达/假设 | 不要求事实证据 | 标题、比喻、结构、节奏、镜头、虚构情境可以自由，但不得伪装成真实顾客、真实效果或品牌承诺 |

发布内容不必展示后台全部证据。系统只需在内部生成“事实主张清单”，逐项验证来源；无法验证的主张应删除、降格为有条件表达或进入人工复核。这样证据是护栏，而不是把全文写成论文。

建议每个内容任务同时形成五个输入包：

1. **事实包**：不可错的商品、品牌、经营和合规事实；
2. **生活包**：真实人物、地点、动作、冲突、细节和经授权的原话；
3. **品牌包**：品牌世界观、判断、声纹、禁区和取舍；
4. **趋势包**：当前平台形式、话题、节奏及其饱和度、风险和有效期；
5. **创意边界**：哪些维度锁定、受限或自由。

## 内容风格策略如何分层

| 层级 | 应进入的内容 | 不应进入的内容 |
|---|---|---|
| 领域通用层 | 内容目标、受众需求、生活场景、叙事视角、冲突与情绪变化、媒介能力、互动机制、创作策略、反模式和评测维度 | 某品牌固定声纹、固定文案模板、实时热点和流行梗 |
| 品牌私有层 | 世界观、价值判断、目标客群生活理解、角色声纹、标志性母题、喜欢/禁止表达、优秀及失败样例、内容组合策略 | 其他租户的私有表达与案例 |
| 区域/门店生活层 | 真实人物、城市天气与通勤、商圈、节庆、门店事件、试穿与服务片段、顾客高频问题 | 未经授权身份信息和虚构成真实的故事 |
| 动态趋势层 | 平台、形式、话题、受众、增长/衰退、饱和度、地域、版权与风险、观测和失效时间 | 自动升级为长期领域知识的热点 |
| 任务运行层 | 当前目标、受众、人物、场景、商品、创意路线、趋势态度、商业引导和验收方式 | 跨任务永久固化的临时创作选择 |

因此，内容风格策略需要进入领域通用知识，但进入的是“创作策略的语义与能力框架”，不是一套统一风格。品牌选择什么、某位店长怎么说、这周是否借势某个趋势，分别由品牌、角色/生活和动态层决定。

趋势只有三种处理结果：品牌化借势、反向表达或忽略。趋势不是默认命令；盲目追随反而会加剧同质化。

## 如何支持不同企业的多样化需求

必须先纠正目标：SaaS 不可能无边界满足所有企业的所有内容需求。正确设计是覆盖稳定高频能力，并提供受控扩展，而不是为每家企业复制系统。

采用三层机制：

1. **稳定内核**：事实、证据、授权、租户隔离、内容任务、输出版本和评测协议保持统一。
2. **可配置内容产品**：每个企业组合自己的业务目标、受众、生活场景、角色、渠道、媒体形式、证据政策、创意自由度、趋势态度、商业引导、审核流程和指标。
3. **受控扩展**：主题、语气、尺寸等差异继续配置；只有输入、输出、业务目标或工作流实质变化时才增加新内容产品；只有新系统接入或全新执行流程才需要插件/定制开发。

一个内容产品可以表示为：

`业务目标 + 受众需求 + 生活场景 + 表达角色 + 渠道/媒体 + 证据策略 + 创意边界 + 趋势策略 + 审核流程 + 成效指标`

这比维护无穷脚本模板更能支持企业差异，也避免品牌私有需求污染领域通用层。

每一项新企业需求必须先经过适配判定，而不是直接扩本体或开代码分支：

| 适配结果 | 判断条件 | 处理方式 |
|---|---|---|
| 直接支持 | 现有内容产品、输入和流程已经匹配 | 直接创建任务 |
| 配置支持 | 只改变品牌、受众、场景、角色、渠道、风格、审核或指标 | 新建租户配置，不改公共代码 |
| 受控扩展 | 业务目标、关键输入/输出、验收合同或外部工作流确实不同 | 新增内容产品、连接器或插件，并保持版本化接口 |
| 明确不支持 | 违反事实、授权、隐私、平台规则，或超出产品战略边界 | 明确拒绝或另行立项 |

立项前不应试图枚举“所有企业需求”。应从若干经营模式和内容成熟度明显不同的候选品牌收集真实任务，形成“需求—内容产品—能力—适配结果”矩阵；只有跨品牌重复出现或对目标试点至关重要的需求，才有资格进入第一版公共能力。单一模拟品牌只能验证数据结构和流程，不能证明多企业适配性。

## 反同质化生产流程

1. 先生成多个结构真正不同的创意路线，例如人物驱动、生活冲突驱动、商品取舍驱动或地方事件驱动，而不是只换同一文案的形容词。
2. 选定路线后再生成草稿；事实包只锁定真相，不预写全部句子。
3. 从草稿抽取事实主张并做证据、授权和合规检查。
4. 与同品牌近期内容比较主题、语义、开头、结构、结尾、人物、场景、情绪和镜头重复。
5. 进行品牌替换测试：替换品牌、商品和城市后仍完全成立，说明内容过于通用。
6. 进行生活具体性测试：是否存在真实行动、时间地点、生活摩擦、细节和有代价的选择，而不是抽象价值宣讲。
7. 进行隐藏品牌名的识别测试：目标受众能否从观点、取舍和声纹辨认品牌。
8. 真实目标受众评审“像不像人说话、是否与我的生活有关、是否愿意继续看”；模型评分只能辅助。

多样性不仅检查单条内容，还应检查一段时间内的内容组合。持续监控主题、角色、商品、区域、媒体形式、叙事路线和情绪的覆盖与连续重复。高播放、高成交只能形成候选策略，不能自动成为事实正确或值得全品牌复制的证据。

还有一条必须纠正：“一眼看不出 AI”不能被翻译成规避生成内容声明或标识。项目可以追求自然、具体、有生活来源和不模板化，但必须保留生成来源、人工改编、发布声明以及适用的显式/隐式标识能力。

## 关键关系

- 租户包含组织主体和品牌；组织主体之间表达拥有、运营、加盟、授权和数据控制。
- 品牌拥有表达档案和内容角色。
- 知识陈述描述商品、概念、组合或情境；证据记录支持或反驳知识陈述。
- 商品款具有可售变体；商品组合包含商品并适用于穿着或陈列情境。
- 真实生活记录发生于人物、门店、时间和地点，并由媒体资产记录。
- 权利许可授权媒体资产或生活记录用于指定渠道和期限。
- 趋势信号提示某主题或形式，但不直接修改品牌或领域知识。
- 内容产品定义稳定的目标、输入、输出、证据政策、创意边界、工作流和验收合同；品牌为其增加租户配置。
- 内容任务书实例化一个内容产品，使用商品、组合、真实记录、趋势和媒体资产，并受策略规则与创意边界约束。
- 内容成品完成任务书、引用知识陈述和证据、使用已授权资产。
- 内容组合策略约束一组内容成品；评测信号评价单条内容或整体组合。

“适合、流行、体现、可搭配”等主观关系不得成为无条件硬边，必须通过知识陈述携带条件、证据、适用范围和审核状态。

## 首批必须回答的业务问题

1. 某项商品或效果主张是否有足够证据，当前可否发布？
2. 当前任务哪些内容必须照实、哪些受限、哪些可以自由创作？
3. 总部设计师与加盟店长如何保持品牌一致，却不说成同一个人？
4. 某个门店故事是否真实，是否拥有当前渠道和期限内的使用许可？
5. 某趋势是否适合当前品牌、地区和内容组合，还是只会造成盲目跟风？
6. 近一段时间内容是否在人物、商品、形式、场景和叙事上过度重复？
7. 不同企业如何配置不同内容产品，而不修改通用核心和复制代码？
8. 当前库存、天气和活动变化后，搭配、陈列或内容建议是否应失效？
9. 任一内容能否追溯到商品事实、真人记录、授权资产、品牌规则和评测结果？
10. 哪些反馈可以改进内容策略，哪些只能作为观察而不能升级为知识？

“材料、版型、色彩、场合、渠道、格式”等受控词汇仍然需要，但它们是支撑语义模型的词表/分类结构，不占一个核心业务对象名额；否则会出现最关键的“内容产品”反而无处建模的倒置。

## 外部研究与现行规则参考

- Science Advances 的短篇故事实验观察到生成式 AI 可能提高单篇作品评价、同时降低作品集合的差异性。该结论不能直接外推为中国自媒体的确定规律，但支持将“作品集同质化”设为需要实测的风险：https://doi.org/10.1126/sciadv.adn5290
- NAACL 2024 的长文本检索增强生成研究发现，检索材料存在并不等于生成句子都有材料支持，因此本项目保留成稿后的事实主张抽取与核验：https://aclanthology.org/2024.findings-naacl.100/
- 《人工智能生成合成内容标识办法》已于 2025-09-01 施行，要求相关服务与传播活动按适用情形处理显式标识、隐式标识和用户声明，因此项目不能以“隐藏 AI 来源”为质量目标：https://www.cac.gov.cn/2025-03/14/c_1743654684782215.htm
