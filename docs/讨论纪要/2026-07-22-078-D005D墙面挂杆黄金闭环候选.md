## 一、这次只证明什么

证明一名门店人员只提交本次实际可用库存后，系统能够复用已有品牌标准、商品知识和挂杆条件，交付一份可直接照做的墙面挂杆文字方案；门店自然反馈一项实际差异后，系统只修订受影响部分。

不证明真实门店已经执行、陈列效果良好、销售提升、DIFY 工作流完成、ERP 已接入或系统具备识图巡检能力。

V0.1 虽然证明了“库存输入—文字方案—自然反馈—局部修订”的语义回路，但冻结的是单层左右排列，且主要按品类和容量组织，不能作为视觉陈列黄金模板。V0.2 不重开 D-005-A—C，只补齐执行成立所必需的垂直结构、动线和搭配组。

## 三、交给门店的首次方案

### 这版怎么挂

二十件不用全部上墙。这版挂十五件，分成三个上下对应的搭配区。顾客主要从左侧通道进入，因此把深绿细格 ZX-C218 放在上杆左侧主正挂；右侧再用同款炭灰面作较弱回应。暖白和燕麦放在两者之间，深绿细格靠明暗、纹理和留白跳出来，不额外编造一件亮色商品。

### 上下、左右和三个搭配区

站在顾客面对墙的位置，从左向右执行：

| 搭配区 | 上杆 | 下杆 | 视觉与穿着关系 |
|---|---|---|---|
| A：入口主焦点 | 左侧固定正挂：ZX-C218 ×1，深绿细格纹面朝外 | 与其垂直对应：ZX-P211 炭灰直筒裤 ×2 | 顾客从左侧进入时先看见格纹外套，再读到炭灰下装；这是全墙最强重点 |
| B：中间基础区 | 两个正挂点之间：ZX-S104 暖白衬衫 ×2 → ZX-K126 燕麦薄针织 ×2 | 与其垂直对应：ZX-Q117 深橄榄中长裙 ×2 | 用浅色上装留出呼吸，并把主焦点的绿色带回一组可穿关系 |
| C：右侧回应区 | 右侧固定正挂：ZX-C218 ×1，炭灰面朝外；其外侧 ZX-V113 炭灰短马甲 ×2 | 与其垂直对应：ZX-P211 炭灰直筒裤 ×1 → ZX-Q117 深橄榄中长裙 ×2 | 让受众发现两处正挂是同一件商品的两面；力度低于入口主焦点，并以两组下装完成搭配选择 |

本版使用十五件：ZX-C218 2 件、ZX-S104 2 件、ZX-K126 2 件、ZX-P211 3 件、ZX-V113 2 件、ZX-Q117 4 件。其余五件保持未上墙。库存清单是选择范围，不是填满墙面的命令。

### 正挂、侧挂、数量与间距

- 两个正挂都在上杆：入口左侧为主焦点，右侧只作同款另一面的回应，不把两处做成同等强度；
- 每个下杆小组与上方搭配区垂直对应，不把裤、裙集中成脱离上装的库存区；
- 同一小组内衣架方向一致、便于抽取；搭配区之间留约一个衣架宽的停顿；
- 主正挂周围保留约一个衣架宽，右侧回应正挂周围保留约半至一个衣架宽；
- 上杆侧挂不超过六件、下杆不超过八件；本版分别为六件和七件，不为达到容量上限继续补货；
- 如果实物厚度或衣袖使相邻商品不好抽取，先减少相邻侧挂，不移动已确认的上下杆和主焦点。

### 替代办法

1. 一件已上墙的 ZX-C218 临时不能使用时，先使用清单中第三件、尚未上墙的同款替换；若门店更正为本次实际只剩一件，则优先保留左侧主正挂并取消右侧回应正挂，不用其他绿色商品冒充同款另一面。
2. 暖白衬衫临时少一件时，先使用清单中第三件、尚未上墙的同款；若同款也不可用，才用一件未上墙的 ZX-K126 燕麦薄针织补入中间基础区，仍保持浅色块，不跨区调用无关商品。
3. 深橄榄半裙临时少一件且清单中没有同款后备时，相应下杆小组少挂一件并重新拉开，不把上杆商品移到下杆凑数量。
4. ZX-C218 全部不能使用时，主焦点和双面回应关系同时失效，应按更正后的清单重新生成，不保留本版标题式概念。
5. 主要来客方向临时改变时，应把它作为新的现场条件告诉系统，主焦点随动线重排；不能只机械镜像文字而不看固定正挂位置。

### 执行步骤

1. 清空本组上下挂杆，按清单分出十五件上墙商品和五件未上墙商品。
2. 先完成上杆两个正挂：左侧 ZX-C218 深绿细格面为主，右侧 ZX-C218 炭灰面为回应。
3. 在上杆中间挂暖白衬衫和燕麦薄针织，在右侧回应点外侧挂两件炭灰马甲。
4. 按 A、B、C 三个垂直搭配区完成下杆裤裙，不把下装重新集中成一个品类区。
5. 统一衣架方向，拉开搭配区与正挂周围的停顿；先保证主焦点清楚和商品便于抽取，再决定是否保留全部十五件。

这是南城店可以采用的内部执行建议，不表示总部批准、系统核验或门店已经完成。

## 四、门店执行后的自然反馈

> 已经照这版挂了。左侧主焦点和下杆都没问题；右上正挂的 ZX-C218 袖子压到旁边两件马甲，最靠近的一件不好拿。其他位置、动线和库存都没变。

系统只把这段话作为南城店本次提交的新条件，不把它改写成“巡检发现不合格”。

## 五、交给门店的修订版

只改 C 区上杆，A、B 两区和全部下杆不动：

- 右侧回应正挂 ZX-C218 保留；
- 其外侧 ZX-V113 炭灰短马甲由两件减为一件；
- 剩余马甲向右重新居中，与正挂外套之间留约一个衣架宽；
- 取下的一件马甲保留为未上墙商品，不转移到其他区，也不用后备商品补满；
- 整墙由十五件变为十四件，主焦点、回应点、上下搭配关系和左向右动线全部保持不变。

门店执行只需取下靠近正挂的一件马甲，再调整剩余一件的位置。反馈没有影响其他区域，因此不重排整面墙。

## 六、最小验收结论

| 验收问题 | V0.2 结果 |
|---|---|
| 是否同时回答上下与左右 | 是；明确上杆、下杆及三个垂直搭配区 |
| 是否为视觉陈列而非容量墙 | 是；两个正挂形成主焦点和回应点，十五件库存按搭配关系选择而非按品类塞满 |
| 是否考虑顾客动线 | 是；主焦点置于已记录的左侧来客入口，动线变化必须更新条件 |
| 低饱和是否仍有可见焦点 | 是；依靠深浅反差、唯一格纹、入口位置和留白，不编造亮色商品 |
| 日常是否只提交库存清单 | 是；上下结构、容量、动线、品牌标准和商品知识均从已记录信息调用 |
| 是否使用对内组织身份 | 是；没有发布账号、IP 人设、平台文案或 CTA |
| 反馈后是否只改受影响部分 | 是；只减少 C 区上杆一件马甲，其他上下关系无损继承 |
| 是否引入巡检、评分或整改 | 否；只有首次建议、自然反馈和修订建议 |

快速复位、顾客翻乱后的维护频率和复位标准具有真实门店价值，但不属于这个首期最小语义闭环；本次关闭不得被表述为已覆盖终端陈列全部价值。

用户已确认 V0.2 黄金闭环成立，并指出唯一残留是黄金视线区高度属于南城店挂杆档案的固定属性而非方案缺陷；运行档案已明确上杆商品主体约位于 120—160 厘米主要视线带，因此不新增字段或测试。D-005-D 与整个 D-005 正式关闭。关闭只证明第一版对内墙面挂杆的语义合同和文字执行闭环成立，不证明真实门店效果；达到该边界后停止扩展。
