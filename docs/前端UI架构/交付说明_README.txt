========== V1.1（当前版本）==========

〇、V1.1 修订依据与内容
依据《笛语前端UI最小实施修订规范_v1.0.md》（创始人六项纠偏 + 三项补丁）成套修订：
1. 动作语义（ADR-026）：取消"采用此版"全局主动作；改为 继续改/复制/导出 + 多版本时按需"设为当前版本" + 明确"存为素材/纳入系列"；"已采用"徽标全部移交"当前版"。
2. 文案：删除自证旁白（"底层检索不显示""不伪装进度"等）；黑话全清（成果合同/业务域/租户/经营真值/服务端/下行可用/黄金成品/作用域/长期记忆/内容角色→发布账号+表达身份）；承重边界人话化保留（未成年人/示意非实拍/清单不动库存账/管理员≠维护权）。
3. 移动端：底部「对话/成品(方案)」两段切换；顶部一行身份摘要（点按看全）；成品视图常驻 继续改+复制/导出；不自建触屏选字。
4. 新增 3 个复用异常模式示例态（14 空状态 / 15 失败与重试 / 16 删除确认），13→16 态。
5. 视觉：正式 VI 资产为唯一视觉真源（Medium 字重，官方 SVG 内联，禁 CSS 手绘 Logo）；朱砂红收敛（撤装饰性眉标/镜头用红，留给当前应用与关键确认）；宋体只留品牌与成品标题；发光圈与闪星图标移除；8/9px 小字升至 10px（12px 底线在生产 token 层执行）。
6. 交互修复：追问态选择卡可点击、发送键显示"生成这一版"、"更多"菜单（复制全文/查看版本/存为素材/结束任务）、首次入口不再混入"继续上次"、系列"下一集"置顶。

二、高保真模型操作
- 顶部状态栏：切换 13 个关键产品状态。
- “桌面 / 移动”：切换响应式模型。
- “?”：显示或隐藏设计标注。
- 搜索图标或 Ctrl/Cmd + K：打开全局搜索。
- 键盘左右方向键：前后切换状态。
- 键盘 M：切换桌面/移动。
- 页面内按钮可触发状态跳转或演示提示。

三、已验证

- 桌面/移动切换、设计标注、全局搜索和 Escape 关闭均通过浏览器运行检查。
- JavaScript 语法检查通过。
- 架构 HTML 编码、目录和关键章节渲染通过。
