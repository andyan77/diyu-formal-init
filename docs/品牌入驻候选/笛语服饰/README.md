## 可公开展示的授权边界

用户已确认可在产品演示、客户交流和公开宣传中使用：笛语服饰名称、经其确认的品牌/商品/门店资料，以及基于这些资料生成的文字成品。

不在本候选入口或公开展示授权内：26 条云盘原视频、未确认候选数据、个人私有素材，以及 Windows `陈列搭配/` 下仅属领域通用库的 19 份重复文件。

## M6-1 最低自然确认项

M6-1 的全局最低门槛是：品牌方自然确认当前品牌基线、至少一个真实使用的发布账号及
独立 `ContentRole`，并由一个已登记自然人取得操作资格。确认后才能写入该租户作用域；
本目录不会自行进入系统运行上下文。

少量真实商品事实和一组真实双层墙面挂杆固定档案仍是后续直接消费者需要的资料，但按
渐进入驻原则属于能力局部门槛：未确认商品只限制商品承重的 P1/P2/P5；未确认挂杆只
限制 DM01，不阻断管理员登录、账号规划、普通交流和非商品承重内容生产。不得为了让
M6-1 全绿而把候选商品、视频推断或门店模板自动升级为事实。
