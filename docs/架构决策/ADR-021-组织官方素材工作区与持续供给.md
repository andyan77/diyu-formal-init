# ADR-021：组织官方素材工作区与持续供给

- 状态：已接受
- 接受日期：2026-07-22
- 对应决策：D-018
- 决策母稿：[D-018 组织官方素材工作区极简方案](../讨论纪要/2026-07-22-084-D018组织官方素材工作区极简候选.md)
- 上游边界：[ADR-008](ADR-008-个人私有原始素材隔离与首期禁止共享.md)、[ADR-009](ADR-009-成长期公共素材发布与创意上下文降级.md)、[ADR-012](ADR-012-授权期个人隔离与退出后所属主体接管.md)、[ADR-019](ADR-019-第一版25类最小语义模型与兼容扩展边界.md)、[ADR-020](ADR-020-多品牌共享隔离与经营关系.md)
- 后续适用范围：[ADR-023](ADR-023-组织型IP参考母版与租户拓扑非强制.md)明确总部、区域、门店素材视图只在相应真实组织存在时形成；单店租户不创建空视图

## 背景

总部、区域代理和独立加盟店都需要积累可供本组织后续账号复用的素材，但每店建设物理数据库不可维护，把私人素材投稿或转成组织素材又会重新打开已关闭的私人素材后门，并引入审批、副本、撤回和退出同步流程。

D-018 因此只解决一个问题：怎样让各经营主体主动维护自己的组织素材，同时保持私人素材、退出接管素材、未来公共素材和正式知识的边界不变。

## 决策

### 1. 一套素材底座，三个组织逻辑视图

第一版继续复用 `MaterialAsset`。总部、区域和门店素材区只是依据租户、控制组织、品牌适用范围、使用范围、版本及状态形成的逻辑视图，不为各层级或每家门店建立物理数据库，也不新增 `Workspace` 核心对象。

### 2. 私人和组织两个入口，首期不转换

- 总部、区域和门店的每个有效自然人账号都可以拥有“我的素材”；由本人主动保存，授权有效期间只供本人在自己的任务中选择使用；
- “本组织素材”由获得本组织素材维护资格的人从组织入口直接录入，从创建起即由该组织控制；
- 第一版不提供私人素材转组织素材、管理员进入私人箱挑选、投稿、审批、共享副本或跨层级上提。

同一人即使同时具有普通使用者和组织素材维护资格，也必须从相应入口直接录入；系统不把已经进入私人素材箱的原件一键提升为组织资产。

总部员工基于岗位任务生成和修改的 `ContentItem` 等业务成果仍属于总部组织工作区；这不改变其私人原始素材仅本人可见。私人素材箱属于实际自然人账号，不属于 `PublishingAccount`、岗位人设或组织管理员。

### 3. 一个控制组织，两种最小使用范围

每项组织素材恰有一个控制 `OrganizationalParty`。只有被明确授予该组织素材维护职责的人可以新增、修订、替换或停用；账号管理员、上级组织和渠道合作方不会自动取得维护资格。

第一版只冻结两种业务使用范围：

1. **仅本组织**：默认；
2. **明确下行合作网络**：总部可以明确供当前品牌合作网络使用，区域可以明确供本区域当前合作门店使用。

门店组织素材默认仅本店可用，不自动向总部、区域或其他门店开放。经营关系只证明合作资格，不能单独产生素材读取权；素材声明的范围、品牌适用范围和当前有效合作必须同时成立。

### 4. 组织官方不等于正式事实

“组织官方”只表示该素材由哪个组织控制和复用，不表示内容已经被系统认证。组织素材不会因进入素材区、被多次使用或表现良好而自动成为：

- 品牌全国事实、总部政策或商品参数；
- 正式 `KnowledgeStatement`、`EvidenceRecord` 或 `PolicyRule`；
- 公共素材、跨组织事实或训练数据。

稳定事实或规则确需进入正式知识时，继续走既有知识维护与版本规则。

### 5. 任务时临时组装，不改变来源身份

一次门店任务可以临时使用：

```text
当前范围内明确向下开放的总部组织素材
＋当前范围内明确向下开放的区域组织素材
＋本店组织素材
＋当前使用者主动选择的私人素材
```

各来源保持自己的编号、版本、控制组织和品牌范围，不物理合并、不互相升级。总部和区域不能借任务编译反向查看门店组织素材；任何其他账号不能借同店关系查看某人的私人素材。多品牌任务继续按 ADR-020 显式过滤，不因同属一个租户串用。

### 6. 持续供给采用少量维护人和自然录入

每个确有组织素材需求的经营主体只需指定一至数名素材维护人。维护人用自然语言直接保存值得跨任务或供本组织多个账号复用的资料；系统从当前入口带出组织、品牌和默认范围，只有存在会实质改变可用范围的歧义时才追问一次。

一次性琐事留在当前任务或私人素材；门店介绍、稳定空间特征、长期服务方式、阶段性活动素材和组织确认的本地资料适合进入组织素材。区域代理可以培训采集方法并维护区域自己的素材，但不能进入门店员工私人箱或集中代替所有门店整理原件。

### 7. 生命周期及后续扩展继续分账

组织素材被替换、停用或删除后停止进入新任务；既有 `ContentItem`、`DisplayArtifact`、`GenerationRun` 和最小 `ActivityEvent` 保留必要的素材编号、版本及来源引用，不改写历史成果，也不以审计为名保留隐藏原件。

退出期满素材继续依 ADR-012 进入所属主体接管区；接管不等于组织官方，也不自动检索。成长期公共贡献继续依 ADR-009 形成独立公共版本；它是新模块的加法扩展，不修改既有私人、组织或接管素材的含义。

## 南城店最小语义校验

复用“折线之间”单品牌合作网络：品牌总部、一个区域代理、独立加盟的南城店、一名南城店组织素材维护人和一名普通店员。只检查当前决策直接承诺的路径：

| 路径 | 预期结果 | 校验 |
|---|---|---|
| 总部活动素材明确设为品牌合作网络可用 | 南城店有效账号可调用；没有该明确范围时不可调用 | 通过 |
| 区域素材明确设为本区域合作门店可用 | 南城店可调用，区域外门店不可调用 | 通过 |
| 南城店维护人从组织入口录入雨天服务素材 | 南城店获授权账号可调用；总部、区域和其他门店不能反向查看 | 通过 |
| 任一总部、区域或门店自然人把照片保存到“我的素材” | 只有该自然人本人可选择使用；同组织其他账号、管理员、上下级组织均不可读取。总部员工的岗位业务成品仍独立进入总部组织工作区 | 通过 |
| 同一素材被停用或进入退出接管区 | 停止新任务自动调用；不自动升级为组织官方、知识或公共素材，既有成果只保留最小引用 | 通过 |

结果为 **5/5 通过**，未发现新增核心类别、物理工作区、逐文件复杂权限矩阵或审批状态机的必要。

该校验只证明语义合同自洽，不证明生产数据库、对象存储、检索过滤、真实账号权限、并发安全、媒体处理或真实企业采用已经实现。

## 明确不做

第一版不建设：

- 每店独立数据库或素材副本同步；
- 私人素材转组织素材、投稿审批或管理员读取私人箱；
- 区域集中转录全部门店素材；
- 逐文件复杂 ACL、权限策略引擎、多级审核、评分或素材任务状态机；
- BBS、跨店素材交流或成长期公共素材空功能；
- 自动把组织素材升级为品牌知识、事实、证据或训练数据；
- 图片/视频编辑、匿名化或多媒体审批。

## 影响

D-018 正式关闭。第一版现在具有私人素材、组织官方素材、退出接管素材和未来公共素材四条不混淆的来源语义，同时允许总部与区域通过明确范围向下赋能，门店保持本地组织素材不向上穿透。

该决策冻结的是业务语义与最小可见边界，不表示素材管理界面、API、物理表、DIFY 编排或生产权限系统已经完成。下一项进入 D-020，裁决亲子业态真人儿童素材是否进入首期门店采集。
