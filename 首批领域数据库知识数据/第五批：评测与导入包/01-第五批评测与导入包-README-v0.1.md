## 4. 导入纪律

- 运行治理生命周期统一为 `review_candidate / active / deprecated`：本包的 `review_candidate` 只表示可进入候选区，不表示已激活生产；`active` 需有直接消费者且当前有效，`deprecated` 可保留历史引用但不得进入运行上下文。
- 导入时先注册来源，再导入资产，再注册夹具并校验交叉引用。
- H 规则是逻辑合同，不要求一条规则对应一个服务、Agent 或 LLM 调用。
- 外部标准和论文 URL 沿用第二批来源索引，本批未重新联网核验。
- 8 个 `G-*` 黄金引用只注册为外部项目夹具桩，不在本包猜补内容。
- 正式激活前仍需按专业复核、黄金任务和真实制作样本逐批确认。
