### A-TERM-001｜纤维—纱线—面料—成衣是不同对象层级

| 项目 | 内容 |
|---|---|
| 需求目录 | `A01` |
| 主题 | `textile_terms` |
| 硬度 | 🟥 确定性知识 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P1 帮助选择；P2 解释商品；P5 创造欲望 |
| 状态 | `review_candidate` |

**知识断言**  
纤维、纱线、平面纺织结构（面料/织物）和成衣属于不同层级；上一层的存在不能替代下一层的结构与成品事实。

**适用范围**  
服装与一般纺织品的基础术语

**定义/测量口径**  
按 ASTM 纺织与织物术语区分材料基本单元、线性集合、平面结构和最终服装制品。

**允许表达**  
`assertive_within_scope`

**禁止误用**  
不得仅凭纤维名称推断整件面料或成衣的全部性能；不得把通用术语冒充具体商品已采用的结构。

**依据引用**  
`STD-ASTM-D4850`

**复核策略**  
标准版本变化时复核；无变化时每24个月抽样复核

### A-MAT-001｜保暖是材料与服装系统共同形成的结果

| 项目 | 内容 |
|---|---|
| 需求目录 | `A02` |
| 主题 | `thermal_behavior` |
| 硬度 | 🟨 条件性专业规律 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P1 帮助选择；P2 解释商品 |
| 状态 | `review_candidate` |

**知识断言**  
当材料结构能够形成稳定空气层、减少热传递并与穿着环境匹配时，服装可能更保暖；纤维名称本身不足以决定实际保暖。

**何时使用**  
比较具体服装或面料组合的热感与保暖预期

**可能效果**  
可能提高热阻并减少体表散热

**为什么**  
热阻受厚度、体积、织物结构、含湿状态、风、里料、松量和多层组合共同影响

**代价/取舍**  
更保暖往往伴随体积、重量、闷热、活动受限或室内过热等代价

**不适用/需谨慎**  
高风环境但外层不防风；潮湿压缩后空气层减少；只知道纤维成分而不知道结构

**禁止误用**  
不得写成“羊毛一定保暖”“克重越高一定越暖”；测试热阻也不等于保证个人舒适。

**依据引用**  
`STD-ISO-11092`; `PAPER-TEYEME-2020`; `PAPER-VESELA-2018`

**复核策略**  
出现黄金任务失败、真实反例或专业争议时立即复核；至少每12个月专家抽样复核

### A-MAT-003｜空气渗透率主要受孔隙与结构影响，不由纤维名单独决定

| 项目 | 内容 |
|---|---|
| 需求目录 | `A02` |
| 主题 | `air_permeability` |
| 硬度 | 🟨 条件性专业规律 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P1 帮助选择；P2 解释商品 |
| 状态 | `review_candidate` |

**知识断言**  
当织物孔隙更开放、组织较疏、涂层和紧密整理较少时，空气通常更容易穿过；纱线、密度、结构和后整理可使同一种纤维表现差异很大。

**何时使用**  
需要解释闷、透风、防风或空气交换时

**可能效果**  
空气交换可能增强，也可能降低防风性

**为什么**  
空气通过路径由孔隙率、织物组织、纱线、密度、厚度和整理共同决定

**代价/取舍**  
高空气渗透可能伴随防风下降；低空气渗透不必然意味着高热阻或低水蒸气传递

**不适用/需谨慎**  
存在膜、涂层、复合结构或未知整理；仅知道“棉/涤纶/羊毛”等纤维名

**禁止误用**  
不得把“棉=透气”“聚酯=不透气”作为商品结论。

**依据引用**  
`STD-ISO-9237`; `PAPER-ADAMEK-2021`; `PAPER-KIM-2021`

### A-MAT-004｜悬垂由重量、弯曲、剪切、结构和后整理共同决定

| 状态 | `review_candidate` |

**知识断言**

**何时使用**  
解释挺括、轮廓保持和结构感

**不适用/需谨慎**

不得把“厚=挺”“轻=软”“双面=更硬”直接写入商品认知。

### A-FIT-002｜动作舒适取决于局部松量、结构、弹性和动作类型
