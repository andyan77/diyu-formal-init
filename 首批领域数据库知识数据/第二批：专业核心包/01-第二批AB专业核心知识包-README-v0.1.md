## 使用纪律

- 本批是**专业知识候选包**，不是已经激活的生产真源。
- 一张卡只保留一个主要硬度；混合断言已经拆卡。
- 确定性卡只在明示口径和范围内确定表达。
- 条件性卡必须同时保留适用条件、可能效果、原因、代价和不适用情形。
- 创意候选只提供方向与变化轴，不得覆盖品牌审美，也不得冒充行业真理。
- 具体品牌、SKU、设计原意、商品参数、库存、价格、门店、真人和动态趋势不进入本包。
- 外部标准和论文只支撑相应定义或关系，不自动把研究结果扩大为所有服装商品结论。
- `review_candidate → active` 必须经过来源复核、服装专业复核和 P1/P2/P5 黄金任务验证。
- 本包不冻结数据库表、本体类、对象身份、DIFY 节点、模型提示词或平台媒体规格。

## 本批最重要的纠偏

1. “羊毛保暖”“棉透气”“短款显高”“宽松遮肉”等均不能作为无条件行业硬事实。
2. 单位面积质量、厚度、挺括、热阻、空气渗透率和水蒸气阻力必须分开。
3. 人体尺寸、成衣尺寸和松量必须分开。
4. 品类名只提供检索线索，不证明具体性能或场合适用。
5. 商品特征可以产生条件性专业解释，但不能自动产生品牌设计动机。
6. P5 的“单一可见造型命题”约束成品组织，不等于固定镜头、固定商品数或固定拍法。
7. 通用知识只帮助系统推理；最终结论仍需结合品牌、商品、账号、受众、任务和制作条件编译。

## 对齐方式

后续复核可直接按知识卡 ID 提出：

- **同意激活候选**；
- **保留但改写**；
- **降级硬度**；
- **拆分**；
- **删除**；
- **需要外部专家/试验补证**。

不需要逐句重读整包，也不把卡片数量作为建设绩效。
