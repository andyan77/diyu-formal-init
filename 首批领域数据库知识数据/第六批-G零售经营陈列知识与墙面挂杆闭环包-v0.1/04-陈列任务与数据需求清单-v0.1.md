# 陈列任务与数据需求清单 v0.1

## 1. 分层总览

| 层 | 保存什么 | 复用范围 | 首期责任 |
|---|---|---|---|
| 系统通用 G 知识 | 焦点、分组、正侧挂、密度、替代、局部修订等无品牌方法 | 全租户按任务调用 | 平台维护，专业复核 |
| 品牌陈列标准 | 品牌当前默认目标、视觉层级、色彩/纹理倾向、密度偏好和禁区 | 当前品牌 | 品牌维护人确认 |
| 商品知识 | 具体商品身份、颜色、图案、品类、双面、结构和其他当前资料 | 当前品牌 | 品牌商品资料责任方 |
| 门店挂杆固定档案 | 上下结构、固定正挂、容量规则、主要来客方向、硬限制 | 指定门店/挂杆组 | 门店首次说明，变化时更新 |
| 本次人工输入 | 实际可用商品、数量、特殊目标和临时限制 | 当前任务 | 门店输入并复核 |
| 方案与版本 | 首次文字方案及修订版 | 当前门店业务工作区 | 系统生成，门店决定是否执行 |
| 自然反馈事件 | 用户执行后的差异与新条件 | 当前任务/版本链 | 门店自报，不作为系统巡检 |

## 2. 任务身份

首期逻辑上至少解析：

- `tenant_id`
- `brand_id`
- `represented_organization_id`
- `workspace_id`
- `store_id`
- `actual_operator_id`
- `application_id = display-merchandising`
- `display_contract_id = DM01_wall_hanging_rail_execution`

对内成果不强填：

- `PublishingAccount`
- `ContentRole`
- `primary_content_product`
- 平台、CTA、发布状态。

## 3. 品牌标准最小内容

- 当前标准版本；
- 默认任务目标；
- 主焦点与次级回应原则；
- 色彩、纹理、轮廓和留白的品牌方向；
- 密度偏好；
- 明确禁区；
- 生效/失效或复核信息。

这不是全行业真理。品牌未确认的系统建议不得静默成为正式品牌标准。

## 4. 商品资料最小内容

对当前任务实际使用的商品，按需要调用：

- 商品引用与稳定身份；
- 品类；
- 当前颜色/图案；
- 可见表面或双面关系；
- 轮廓/长度/厚度/袖量等与陈列有关的已知特征；
- 本次可用数量；
- 不能被系统创造的商品事实。

通用 A/B/G 知识只能解释和组合，不能替代具体商品资料。

## 5. 门店挂杆档案

逻辑字段候选：

```yaml
store_id:
display_zone_id:
display_carrier_id:
structure:
  upper_rail:
  lower_rail:
  fixed_front_facing_points:
capacity_rules:
primary_approach_direction:
main_sight_band:
hard_constraints:
last_confirmed_at:
version:
```

字段名和存储方式留给 D-016。用户界面只呈现自然语言摘要与必要纠正入口。

## 6. 本次库存输入

```yaml
inventory_input:
  source: user_provided
  scope: current_task
  lines:
    - product_ref:
      quantity:
```

禁止：

- 标记为 ERP verified；
- 自动写回长期库存；
- 建立来源排名；
- 生成新鲜度分数；
- 扣减库存；
- 把未上墙数视为库存余额真值。

## 7. 反馈与修订输入

```yaml
feedback:
  source: user_provided
  original_text:
  changed_conditions:
  explicitly_unchanged:
  affected_scope:
```

系统可以从原文中提取结构化摘要，但必须保留原意，并允许用户纠正。

## 8. 信息不足处理

| 缺口 | 处理 |
|---|---|
| 首次没有容量/结构，且用户要数量间距方案 | 只问一个关键问题 |
| 没有特殊目标 | 使用品牌当前默认标准 |
| 没有品牌标准 | 降级为通用候选，不冒充品牌正式方案 |
| 某个非关键商品缺失 | 同款后备 → 职责相近替代 → 少挂 |
| 主焦点商品全部缺失 | 重新生成主线 |
| 不知道现场是否已经执行 | 保持建议口径 |
| 用户要求识图巡检/评分 | 说明首期不支持，转为自然反馈文字修订 |
