## B08｜可见造型命题与观看关系

### B-VIS-001｜可见造型命题必须能被具体复述，而不是只剩抽象风格词

| 项目 | 内容 |
|---|---|
| 需求目录 | `B08` |
| 主题 | `visible_styling_proposition` |
| 硬度 | 🟨 条件性专业规律 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P5 创造欲望；P1 帮助选择；P3 建立人格；P4 经营关系 |
| 状态 | `review_candidate` |

**知识断言**  
一条可见造型命题应让受众能够具体说出商品、身体、动作或环境之间出现了什么新的穿着关系；“高级、松弛、氛围感、秋日感”等抽象词本身不足以构成命题。

**何时使用**  
系统需要判断当前视觉方向是否已经从审美形容词收敛为可拍、可看、可复述的造型关系

**可能效果**  
让商品关系、画面入口和受众获得的穿着可能保持一致

**为什么**  
具体关系可以被画面验证、被商品替换测试，也能约束后续动作、构图和文字

**代价/取舍**  
围绕一条命题会主动放弃同篇展示更多卖点或搭配可能

**不适用/需谨慎**  
任务主要价值是商品解释、人格观点或本地回应，视觉只作辅助；或尚无法建立真实品牌商品身份

**禁止误用**  
不得把抽象情绪词、品牌口号、流行滤镜或普通站姿包装成 P5 命题；不得要求所有视觉内容都采用同一种二元对比结构。

**依据引用**  
`PRJ-P5-CLOSE-20260721`；`PRJ-ADR-013`；`PRJ-AB-B08`；`EXP-BE-VISUAL-001`

**复核策略**  
出现 P5 黄金/回归失败、商品或品牌替换后不敏感、视觉模板化上升或专业争议时立即复核；至少每12个月结合真实样本抽样复核

**验证夹具**  
`BE-V01`；`BE-V08`

### B-VIS-002｜商品数量由造型命题的必要关系决定，不由“搭配完整度”决定

| 项目 | 内容 |
|---|---|
| 需求目录 | `B08` |
| 主题 | `necessary_product_relationship` |
| 硬度 | 🟨 条件性专业规律 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P5 创造欲望；P1 帮助选择；P2 解释商品 |
| 状态 | `review_candidate` |

**知识断言**  
当一件商品与身体、动作或环境的关系已经足以让穿着可能成立时，不应为了看起来更完整强迫加入鞋包、配饰或第二件服装；只有新增商品承担不可替代的比例、层次、色彩、功能或场景关系时才应加入。

**何时使用**  
系统需要确定单件、两件或多件商品是否都是当前命题成立所必需

**可能效果**  
减少无意义搭配堆砌，使每个商品都有明确视觉职责

**为什么**  
商品数量本身不创造造型价值，关系才创造价值

**代价/取舍**  
更少商品可能降低商品曝光广度，但提升命题清晰度和制作可行性

**不适用/需谨慎**  
任务本身明确需要比较、层搭、组合替代或多商品关系

**禁止误用**  
不得把“单件足够”写成 P5 固定规则；也不得把每份 P5 做成三套穿法或搭配清单。

**依据引用**  
`PRJ-P5-CLOSE-20260721`；`PRJ-ADR-013`；`PRJ-AB-B08`；`EXP-BE-VISUAL-001`

**复核策略**  
出现 P5 黄金/回归失败、商品或品牌替换后不敏感、视觉模板化上升或专业争议时立即复核；至少每12个月结合真实样本抽样复核

**验证夹具**  
`BE-V09`

### B-VIS-003｜造型命题需要由商品与身体、动作、其他单品或环境中的至少一项关系承重

| 项目 | 内容 |
|---|---|
| 需求目录 | `B08` |
| 主题 | `relationship_carrier` |
| 硬度 | 🟨 条件性专业规律 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P5 创造欲望；P1 帮助选择；P2 解释商品；P3 建立人格；P4 经营关系 |
| 状态 | `review_candidate` |

**知识断言**  
P5 的新穿着可能不能只停留在商品静态外观；商品与身体比例、穿着动作、其他单品或环境之间至少一项关系，应真正改变受众怎样看见和想象这件商品。

**何时使用**  
商品本身已经出现，但系统仍需判断画面是否创造了新的穿着想象

**可能效果**  
使视觉内容从“看见商品”推进到“看见怎样穿、怎样动或怎样进入生活”

**为什么**  
穿着想象来自关系，不来自孤立的商品平铺或口号

**代价/取舍**  
关系越具体，越需要真实制作条件和商品忠实度支撑

**不适用/需谨慎**  
P2 只需展示结构细节来解释商品，或任务明确是商品目录/资料整理而非品牌内容

**禁止误用**  
不得把任意环境、道具或动作都视为有效关系；它们必须实质改变视觉主线，不能只是装饰。

**依据引用**  
`PRJ-P5-CLOSE-20260721`；`PRJ-MOTHER-V03`；`EXP-BE-VISUAL-001`

**复核策略**  
出现 P5 黄金/回归失败、商品或品牌替换后不敏感、视觉模板化上升或专业争议时立即复核；至少每12个月结合真实样本抽样复核

**验证夹具**  
`BE-V01`；`BE-V10`

### B-VIS-004｜比较或变化型画面应尽量明确一个主要变量，并保留足够不变量

| 项目 | 内容 |
|---|---|
| 需求目录 | `B08` |
| 主题 | `visual_variable_and_invariant` |
| 硬度 | 🟨 条件性专业规律 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P5 创造欲望；P1 帮助选择；P2 解释商品 |
| 状态 | `review_candidate` |

**知识断言**  
当造型命题依赖前后变化、两面、两种穿法或两种状态时，保持人物、内搭、动作、光线、机位或背景中的足够条件不变，有助于受众看清真正改变的视觉关系；但不是所有 P5 都必须做控制实验式对比。

**何时使用**  
当前视觉价值依赖“哪里变了”或“为什么看起来不同”

**可能效果**  
增强变化的可辨识度和商品/造型关系的说服力

**为什么**  
同时变化过多会让受众无法确定视觉差异来自商品、动作、光线还是剪辑

**代价/取舍**  
保留不变量可能减少场景丰富度和视觉变化数量

**不适用/需谨慎**  
命题由单一完整画面直接成立，或多场景本身就是受众要看见的穿着进入方式

**禁止误用**  
不得把动作对位、硬切或前后对比固化为所有 P5 的固定模板；不得用不同角度、滤镜和姿势伪造商品效果。

**依据引用**  
`PRJ-P5-CLOSE-20260721`；`PRJ-ADR-014`；`PRJ-CDF-D`；`EXP-BE-VISUAL-001`

**复核策略**  
出现 P5 黄金/回归失败、商品或品牌替换后不敏感、视觉模板化上升或专业争议时立即复核；至少每12个月结合真实样本抽样复核

**验证夹具**  
`BE-V03`；`BE-V11`

### B-VIS-005｜品牌审美应实质改变视觉重音、空间、动作或节奏，而不是只加标识

| 项目 | 内容 |
|---|---|
| 需求目录 | `B08` |
| 主题 | `brand_visual_structural_sensitivity` |
| 硬度 | 🟩 审美/创意候选 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P5 创造欲望；P3 建立人格；P4 经营关系 |
| 状态 | `review_candidate` |

**知识断言**  
同一商品与受众任务在不同品牌战略下，视觉主次、人物状态、空间距离、动作幅度、停留时间和画面秩序应可能发生结构性变化；品牌名、Logo、固定色调或滤镜不能单独证明品牌视觉成立。

**创意方向**  
用品牌已经确认的选择与拒绝、关系姿态和审美边界，决定什么先被看见、什么被留白、人物怎样进入画面以及镜头怎样停留

**变化轴**  
- 视觉重音
- 空间密度
- 人物距离
- 动作幅度
- 停留时长
- 光线与材质关系
- 环境进入方式

**反套路提醒**  
不得把所有品牌都编译成低饱和、慢镜头、自然光和所谓高级感；不得把品牌差异压缩成 Logo、片尾和滤镜。

**禁止误用**  
品牌视觉组织必须来自企业已确认的品牌基线，不能由行业原型或模型偏好静默决定。

**依据引用**  
`PRJ-P5-CLOSE-20260721`；`PRJ-MOTHER-V03`；`PRJ-STRATEGY-V1`；`EXP-BE-VISUAL-001`

**复核策略**  
出现 P5 黄金/回归失败、商品或品牌替换后不敏感、视觉模板化上升或专业争议时立即复核；至少每12个月结合真实样本抽样复核

**验证夹具**  
`BE-V04`；`BE-V12`

### B-VIS-006｜新的穿着可能可以由单帧、动作或序列直接成立，不强制故事情节

| 项目 | 内容 |
|---|---|
| 需求目录 | `B08` |
| 主题 | `non_narrative_visual_legitimacy` |
| 硬度 | 🟩 审美/创意候选 |
| 语义距离 | `commercial_core` |
| 支撑产品 | P5 创造欲望；P1 帮助选择；P2 解释商品；P3 建立人格；P4 经营关系 |
| 状态 | `review_candidate` |

**知识断言**  
P5 可以通过单帧关系、穿着动作、细节—整体序列、静物与人物对照或纯视觉节奏完成；只有人物关系、事件推进或情境变化确实增加穿着想象时才需要故事。

**创意方向**  
优先选择最能让造型关系被看见的视觉结构，而不是默认给每条内容添加人物冲突、情节反转和人生升华

**变化轴**  
- 单帧直接成立
- 动作展开
- 细节到整体
- 整体到细节
- 静物与穿着
- 重复动作中的单变量变化

**反套路提醒**  
不得为了像自媒体内容而硬造故事、对白或反转；也不得把纯视觉误解为没有主线的漂亮镜头拼贴。

**禁止误用**  
调用构图或镜头能力不自动把 P1—P4 改路由为 P5，仍应先看主要产品价值。

**依据引用**  
`PRJ-P5-CLOSE-20260721`；`PRJ-ADR-013`；`PRJ-ADR-014`；`EXP-BE-VISUAL-001`

**复核策略**  
出现 P5 黄金/回归失败、商品或品牌替换后不敏感、视觉模板化上升或专业争议时立即复核；至少每12个月结合真实样本抽样复核
