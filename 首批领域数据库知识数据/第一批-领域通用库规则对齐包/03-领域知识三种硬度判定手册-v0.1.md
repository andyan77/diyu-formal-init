# 领域知识三种硬度判定手册 v0.1

## 1. 适用范围

本手册只用于 A/B/C 及 E 中确属“领域断言”的内容。以下资产不使用三种硬度：

- 创作/制作方法：使用 `method` 合同；
- 行业角色模板：使用 `role_template` 合同；
- 产品与评测规则：使用 `rule_type`；
- 平台趋势：使用动态信号合同；
- 用户种子和品牌专属资料：按其自身作用域管理。

硬度不是重要性、来源等级或法律责任等级，而是“系统允许怎样表达和推理”。

## 2. 三种硬度

| 名称 | 定义 | 生成使用 | 最低记录 |
|---|---|---|---|
| 🟥 `deterministic_fact` 确定性知识 | 在明确术语、范围、测量方法或规范版本内可确定表达 | 可在范围内直接陈述；不得擅自改写定义、数值或规则含义 | 陈述、范围、定义/测量方法、依据、复核信息 |
| 🟨 `conditional_rule` 条件性专业规律 | 结果取决于目标、服装、身体/活动、场景或组合条件 | 必须保留条件、可能效果、代价和不适用情形；避免必然化 | `when / likely_effect / because / tradeoffs / not_when` |
| 🟩 `creative_candidate` 审美与创意候选 | 主观取向、可选审美方向、灵感和经验性操作 | 只能作为候选，不得冒充普遍真理；应提供变化轴和反套路提示 | 方向、可变轴、适用语境、反套路、冲突边界 |

## 3. 判定流程

1. **它是否是方法、模板或规范？** 是：退出硬度判定，进入对应合同。
2. **它能否在明确范围内被定义、测量或由标准术语唯一说明？** 是：倾向 🟥。
3. **它的结果是否依赖目标、结构、组合、活动、环境或个人偏好？** 是：🟨。
4. **它是否主要表达审美、气氛、灵感或可选方向？** 是：🟩。
5. **一句话同时含三类内容？** 拆成多条卡片，不做“混合硬度”单卡。

## 4. 表达边界

### 🟥 确定性知识

允许：

- “服装衣长的测量起点应按本条定义执行。”
- “该洗护符号在所引用标准中表示不可漂白。”

禁止：

- 把适用范围省略后推广为所有产品；
- 把通用定义冒充某个品牌商品已经采用的事实；
- 因模型常识而修改标准术语、单位或测量方法。

### 🟨 条件性专业规律

推荐句式：

- “当……且目标是……时，……通常/可能有助于……；如果……，结果可能相反。”
- “这一判断取决于……，不能只看单一标签。”

禁止：

- “一定、必然、所有人、怎么穿都……”；
- 用一个身材标签直接决定全部答案；
- 只写收益，不写代价和反例。

### 🟩 审美与创意候选

推荐句式：

- “可以尝试……作为本次视觉方向。”
- “一种可选处理是……，也可以沿……变化。”

禁止：

- “秋天就应该穿大地色”；
- “高级感必须来自低饱和”；
- 把平台当前流行拍法写成稳定审美真理。

## 5. 高频误判校准

| 原始说法 | 正确硬度 | 更准确写法 |
|---|---|---|
| 羊毛保暖 | 🟨 | 羊毛纤维及其织物结构常有助于保留空气层，但实际保暖受克重、厚度、组织、湿度、里料和搭配影响 |
| 棉透气 | 🟨 | 棉纤维具有吸湿特性；整件织物的透气性仍取决于纱线、密度、组织、涂层和结构 |
| 双面呢的定义 | 🟥，须绑定口径 | 按本库采用的术语口径说明双面织物/双面呢的结构定义与商业称谓边界 |
| 落肩显随性 | 🟨 | 落肩结构可能弱化肩线并增加松弛感，但结果受肩宽、袖窿、面料和整体量感影响 |
| 短外套显高 | 🟨 | 当衣长落点、上下装比例和视觉重心协调时，短外套可能提高腿部比例感；并非对所有组合成立 |
| 梨形适合 A 型 | 🟨 | 当目标是平衡髋部视觉量感且面料、裙摆宽度和长度合适时，A 型轮廓可能有效；条件不符时可能增加量感 |
| 收腰显曲线 | 🟨 | 收腰结构可能强调腰部与上下围度差，但效果取决于收腰位置、松量和个人目标 |
| 宽松遮肉 | 🟨 | 宽松可以减少贴附，但过大的量感、长度或肩线也可能增加视觉体积 |
| 黑色显瘦 | 🟨 | 低明度可能降低部分区域的视觉突出度，但轮廓、材质反光、比例和背景关系同样关键 |
| 同色系更高级 | 🟩/🟨拆分 | “同色关系减少视觉切割”可作条件规律；“更高级”只能作为审美候选 |
| 秋天适合大地色 | 🟩 | 大地色可以作为秋季视觉方向之一，也可用灰绿、深红或低饱和蓝建立不同温度感 |
| 正式场合必须穿黑色 | 🟨 | 正式度由场合规则、剪裁、材质、整洁度、色彩关系和组织文化共同决定 |
| 真丝都难打理 | 🟨 | 护理难度取决于纤维、织法、染整、结构和洗护说明，不能只按品类绝对化 |
| 纯天然一定更好 | 🟩误作事实 | “天然/合成”不是单一质量等级，应按用途、触感、耐久、护理和成本取舍判断 |
| 厚面料更挺括 | 🟨 | 挺括度与厚度可能相关，但还取决于纤维、组织、后整理和结构支撑 |
| 红色代表热情 | 🟩 | 红色可作为能量、节庆或警觉等多种视觉候选，含义受文化、明度和搭配语境影响 |
| 咖啡、报纸、逆光代表秋天 | 🟩 | 只是常见视觉套路，应作为可选符号并附反套路提示 |
| 竖条纹显高 | 🟨 | 线条方向可能引导纵向观看，但条纹宽度、间距、轮廓和整体切割会改变结果 |

```

```

## 7. 来源与复核要求

- 🟥：需要可靠定义、规范、测量方法或可复核专业依据；“模型知道”不能代替依据。
- 🟨：需要说明推理基础，可以来自可靠资料、专业共识或经确认的专家方法；必须保存条件和反例，不要求伪造精确置信分。
- 🟩：不要求证明审美真理；重点是标明其候选性质、变化轴和品牌冲突边界。

来源要求不能反向扩张到用户本次生活种子。用户种子是当前任务前提，不进入领域通用知识硬度判定。

## 8. 生成器调用规则

| 硬度 | 检索后注入方式 | 输出检查 |
|---|---|---|
| 🟥 | 作为不可擅自改写的范围内前提 | 是否改变定义、数值、单位或适用范围 |
| 🟨 | 连同条件、代价和反例一起注入 | 是否遗漏条件、写成必然、越过不适用边界 |
| 🟩 | 作为可选方向与变化轴注入 | 是否冒充行业真理、重复套路或覆盖品牌审美 |

## 9. 快速复核清单

- 这条内容是知识断言，还是方法/模板/规范？
- 结论是否依赖目标、身体、服装、场景或组合？
- 是否把“通常”写成“必然”？
- 是否把品牌商品事实偷换成通用知识？
- 是否把审美词当作客观功能？
- 是否写明不适用条件和代价？
- 是否能被 P1—P5 的真实任务消费？
- 如果删除这条知识，成品判断或质量是否真的会变差？
